## Tools and services

Xicar uses the following services:

- **GitHub** for source code and collaboration
- **Bitwarden** for shared credentials and secrets
- **Firebase** for application services and distribution
- **Hostinger** for hosting

Access to these services is invite-only. Ask your team lead or project owner for the invitations you need.
